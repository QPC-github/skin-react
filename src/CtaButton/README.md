# CtaButton

## CtaButton Usage

```marko
<CtaButton href="http://www.ebay.com">text</CtaButton>
```

## CtaButton Attributes

Name | Type | Stateful | Required | Description
--- | --- | --- | --- | ---
`size` | String | No | No | Can be "large". If not set, defaults to default
`href` | String | No | Yes | link target

## CtaButton Events

None
