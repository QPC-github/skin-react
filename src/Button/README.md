# Button

## Button Usage

```react
<Button variant="primary" aria-label={` Button`}>
    Button
</Button>
```

## Button Attributes

Name | Type | Stateful | Required | Description
--- | --- | --- | --- | ---
`type` | ButtonTypes | No | no | 
`size` | any | No | no | 
`variant` | any | No | no | 
`isBorderless` | boolean | No | no |
